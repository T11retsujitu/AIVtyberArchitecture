# チェーホフの銃（最初の変な物が最後に効く）

> 共通原理（1動画1技法・5ビート・変換テンプレ）は `docs/05` が正。

- **一言**：冒頭で提示した要素が、終盤で重要な意味を持つ。提示した要素は必ず使う。
- **情報関係**：双方が不完全（物は両者に見えるが、用途はどちらも知らない）。
- **視聴者の問い**：最初のアレは、どう使われるのか？
- **優先度**：**5位**。冒頭の「変な物」がそのまま hook になり、回収の瞬間が再視聴（「最初から映ってた！」）を促す。

## 25秒での形

| 拍 | 例（部屋に一本だけ傘） |
|---|---|
| 0-2s | 密室。出口なし。中央に**傘が一本だけ**立っている |
| 3-7s | AIちゃんは傘を一瞥して壁を調べる。「……出口、ないなあ」 |
| 8-14s | 床・壁を試す（傘は放置）。視聴者「傘だろ！」 |
| 15-20s | 天井が開き、水が流れ込む——傘を開くと**水流に乗って上へ** |
| 21-25s | 「かさって、こういうときの、ものだっけ……？」 |

## 契約との噛み合わせ（要注意）

- 銃（変な物）は冒頭から `scene.elements` に **salience: 'faint'〜'clear' で置き、ref を固定**する（`docs/07`：同一対象＝同一 ref。回収時に「あの物だ」と分かるのは ref 同一性の仕事）。
- 回収を保証するのは **apply/isTerminal の設計**（傘を使う action が終盤で満ちに繋がる）。演出の気合ではなく機構で回収する。
- 使われないまま終わる take（AIちゃんが最後まで傘を無視）は**未回収＝原則ボツ**（Mode B+ 選定基準）。ただし「使われなかった物が最後に一言だけ拾われる」余韻型は可。
- 冒頭の提示は hook カード（`GameMeta.hook`・`docs/02`）と併用できる：「出口のない部屋に、傘が一本」。

## ゲーム例の種

- **部屋に一本だけ傘**（上記）。
- **氷の部屋のマッチ一本**：使いどころを誤ると即溶ける・正しく使うと出口が融ける。
- **懐中時計を持った眠り猫**：終盤、時計が「戻る」ための道具だと判明。

量産性：**中**。「変な物×真の用途」の組み合わせだが、用途の意外性が毎回必要。

## ヘイト管理メモ

**勝ちも負けもある回**。視聴者が先に用途に気づく（「傘を使え！」）とアイロニー寄りに、AIちゃんが先に気づくと秘策寄りになる——**同じゲームで両方の take が出る**のがこの型の妙。負け take でも「物のせい」ではなく「気づけなかった」なので被害者化は弱い。
