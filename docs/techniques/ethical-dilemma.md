# 倫理的ジレンマ（正解ではなく、どちらを選ぶか）

> 共通原理（1動画1技法・5ビート・変換テンプレ）は `docs/05` が正。

- **一言**：正解が存在しない選択。**両方の選択肢に具体的な利益と損失**を置き、視聴者の価値観を割る。
- **情報関係**：双方が別の情報（正確には、双方が別の**価値観**）を持つ——解釈対立・コメント議論。
- **視聴者の問い**：自分なら、どちらを選ぶか？
- **優先度**：**8位**。コメントは狙えるがテンポ設計が難しく、**多用すると説教臭くなる**。シリーズの一部（アクセント回）に限定。

## 25秒での形

| 拍 | 例（一匹か、五匹か） |
|---|---|
| 0-2s | 電池がひとつ。装置がふたつ |
| 3-7s | 左：目の前で震える小さな生き物一匹を温められる。右：眠っている五匹を起こせる——ただし起きた後が安全かは分からない |
| 8-14s | 「……起きてない子は、まだ、苦しくないのかも」 |
| 15-20s | 左を選ぶ。一匹が息を吹き返す。右の五匹は眠ったまま |
| 21-25s | 「まちがってても、この子の分は、まちがってない……と思う」 |

## 契約との噛み合わせ（要注意）

- 「押すか押さないか」だけでは弱い（既存の態度ゲーに退行する）。**両選択肢の利益・損失を描写で具体化**する——`affordances[].label/hint` と `scene.elements` に両天秤を見せる。
- 選ばれなかった側の結末も**画面（Tier B）に映す**（眠ったままの五匹）——選択の重さは省略しない。ただし残酷にしすぎない（下記ヘイト管理）。
- どちらを選んでも `valence: 'bad'` で罰しない。結末は **neutral／strange＋余韻**が基本（`docs/01` 型2醒め・満ちの中間のような閉じ）。「正解がない」を機構でも守る——**片方が隠れ正解なら、それはジレンマではなくルール発見**。
- 25秒で両天秤を説明し切るのが最大の難所。要素は**2択・各1利益1損失まで**。

## ゲーム例の種

- **一匹か、五匹か**（上記）：トロッコ問題の夢的翻訳。
- **名前をひとつ置いていく部屋**：進むには「記憶」「名前」「眠気」から1つ手放す。
- **傘を貸すか差すか**：濡れている誰かと、自分の眠気（風邪をひくと夢が終わる）。

量産性：**中**。思考実験（外部ネタ帳の鉱脈）から夢的翻訳で量産できるが、投稿頻度は意図的に絞る。

## ヘイト管理メモ

**引き分け回**（勝ち負けなし）。AIちゃんの選択自体がキャラ表現になる最良の型——ただし**彼女の選択を罰しない**こと（選んだ結果が惨事になると「道徳的に責められるAIちゃん」になり、コメント欄が彼女への断罪に変わる）。議論はさせる、断罪はさせない。
