# 心理戦・ブラフ（誰を信じるか）

> 共通原理（1動画1技法・5ビート・変換テンプレ）は `docs/05` が正。

- **一言**：相手の発言の矛盾・口調・振る舞いから、誰を（何を）信じるかを判断する。誤認・圧力・情報選択による誘導。
- **情報関係**：双方が別の情報を持つ（NPC の真意は誰も直接知らない）。
- **視聴者の問い**：信じてよいか？　自分ならどちらの言い分を取る？
- **優先度**：**6位**。会話（言語）で戦う型なので LLM と相性がよい。

## 25秒での形

| 拍 | 例（二つの宝箱） |
|---|---|
| 0-2s | 喋る箱がふたつ。片方に鍵、片方に罠 |
| 3-7s | A「Bは必ず嘘をつく」／B「わたしを開けると爆発する」 |
| 8-14s | 「……Bが嘘つきなら、爆発は、しない……？」推論を声に出す |
| 15-20s | どちらかを開ける |
| 21-25s | 鍵（or 罠）。「箱にも、正直ものと、そうじゃないのが、いるんだね」 |

## 契約との噛み合わせ（要注意）

- **NPC の台詞は perceive() が著述する**（`scene.elements[].description` / `feedback.description`）。RawState（どちらが嘘つきか）から決定論的に生成される固定台詞であり、NPC は LLM ではない。
- **LLM 同士の対話（AIちゃん vs 嘘つき NPC vs 正直だが説明が下手な NPC の三者、Among Us 型）は次 wave 予約**。ループ契約（`docs/09`）が単一 LLM 前提のため、多者対話は契約拡張＝doc-first＋人間承認が要る。本 wave では着手しない（この1行が予約の記録）。
- 論理パズル部分は**2文で完結する矛盾**まで（騎士と悪党パズルの最小形）。3者以上の論理連鎖は25秒で追えない。
- AIちゃんの推論は observation/speech で声に出させる——視聴者が推論を並走できる（＝サスペンスを兼ねる）。

## ゲーム例の種

- **二つの宝箱**（上記）：発言矛盾の最小形。
- **道を教える二匹の鳥**：一羽は昨日から嘘つき。口調と羽の色に手掛かり。
- **「開けないで」と「開けて」のドア**：両方が自分を開けろ／開けるなと主張する。

量産性：**中**。嘘つきパズルの変形で作れるが、論理の複雑度上限が低い（25秒制約）。

## ヘイト管理メモ

**勝ちも負けもある回**。推論が当たれば「賢い回」、外れても「論理は通っていた」が担保されていれば被害者化しない——**後出しの裏切り（手掛かりゼロの嘘）だけは禁止**（fair-twist と同じフェアネス原則）。NPC の嘘は録画を見返すと必ず見抜けた、が合格条件。
